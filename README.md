# todor2kc.github.io
Asymmetry

Ovaj repo sadrzi file-ove za asymetry site i pripadajuce "igrice".

Svaka strana pocev od prve ima dugme koje vodi ka sledecoj. Druga strana "memory game" ima tajmer gde ce se putem alerta nakon isteka tajmera prelazi na narednu stranu.

Radjen je na laptopu od 13" za vece monitore i rezolucije bi trebalo prilagoditi responsive.
